本版本修改重点：
- 页面名称改为“杭州市主城区城市食物环境可视化平台”。
- 坐标说明改为 WGS84 / EPSG:4326。
- 默认打开“食物环境”控制面板，左侧可选择时间、线上/线下、正餐/快餐、四类指标。
- 右侧统计分析面板已隐藏，地图作为主展示区域。
- 删除/隐藏“数值合计”等不适合解释的统计项。
- 热力图开启时默认隐藏小区点位；需要查看点位时可勾选“显示小区点位”。
- 使用关键字或分类筛选后，被检索到的小区会以橙色点位突出显示。
